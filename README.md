## Installing

Installing through the Python Package Index:

    $ sudo pip install packetmq
OR

    $ sudo easy_install packetmq

### Requirements

* twisted >= 15.4.0
* u-msgpack-python >=2.1
* bidict>=0.9.0.post1
* Python 2.7.x
